# JournalLib

### Build lib
```bash
make build_lib
```

### Build lib and example
``` bash
make
```

### Run example
``` bash
make run
```

### Build and test example
``` bash
make tests
```
Uses the input data set from the `test_input.txt` file.

### Clear temp files
``` bash
make clear
```

### Clear all build files
``` bash
make clear_all
```
